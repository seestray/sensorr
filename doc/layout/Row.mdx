## Movies usage
Filled dynamicly with `TMDB` api
<Playground style={{ fontSize: '0.5em' }}>
  <Row
    label="Trending"
    uri={['trending', 'movie', 'week']}
    params={{ sort_by: 'popularity.desc' }}
    child={Film}
    style={{ color: 'white' }}
  />
</Playground>

Filled by `items`, `style` css applied to `h1.label`
<Playground style={{ fontSize: '0.5em' }}>
  <Row
    label="Similar"
    items={fixtures.movie.similar.results}
    child={Film}
    style={{ color: 'white' }}
  />
</Playground>

With `empty` props applied to `Empty` child component
<Playground style={{ fontSize: '0.5em' }}>

    label="Recommendations"
    items={[]}
    child={Film}
    style={{ color: 'white' }}
    empty={{ style: { color: 'white' } }}
  />
</Playground>
